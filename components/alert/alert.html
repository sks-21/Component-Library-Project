Alert

An alert displays a short, important message in a way that attracts the user's attention without interrupting the user's task. It provides contextual feedback and communicates a state that affects a system, feature or page.

Basic Alert

Basic alerts are available for any length of text without a close button. It's available in four types - primary alert, error alert, success alert and warning alert.

To use this component, copy the code snippet, where class alert is used for basic stucture and alert alert-primary, alert-secondary, alert-error, alert-warning for different alert types.
