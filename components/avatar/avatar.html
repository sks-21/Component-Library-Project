Avatar

Avatar can be used to show user's profile picture on profile information page, on navigation bar, in blogs grid items.

Avatar is available in four different sizes. The avatar is the base class and avatar-[size] is used for example avatar-xlg, avatar-lg, avatar-md and avatar-sm for different sizes of avatar.

Image Avatar

Image avatar are used to display user's profile picture.

Text Avatar

Text avatars can also be used instead of any user's picture. The initals of the user acts as it's content

The .bg-[color] like .bg-orange etc can be used to give the background color.
